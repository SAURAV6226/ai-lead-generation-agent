# AI Lead Generation Agent

This project demonstrates an AI-powered workflow that helps agencies generate and qualify leads automatically.

## What it does
- Finds potential leads from online sources
- Filters ideal prospects based on ICP
- Generates personalized outreach messages using AI
- Stores leads in a structured format (Google Sheets / CRM)

## Tools Used
- OpenAI / Claude
- Google Sheets
- Zapier / Make

## Use Case
Built for agencies and service businesses struggling with manual lead generation and low response rates.

## Outcome
- Saves 5–10 hours/week on prospecting
- Improves outreach personalization
- Creates a scalable lead pipeline

## Status
Building initial workflow and testing automation

1. Identify target audience (ICP)
2. Collect leads manually or via scraping tools
3. Use AI to generate personalized outreach
4. Store and track leads in Google Sheets
